Support

Need help with ADAMZ uploads, payment, dashboard analysis, or export workflows?

Contact channels

  • Email: support@adamz-ai.com
  • Response target: within 1 business day.
  • Business hours: Monday to Friday, 9:00 AM to 6:00 PM (local business time).

What to include in your request

  • Transaction ID (for payment or storage-related issues).
  • File type and approximate row count.
  • Exact error message and screenshot.
  • URL where the issue occurred (`upload.html` or `history.html`).

Common support topics

  • Stripe checkout and payment confirmation troubleshooting.
  • Storage API configuration, CORS, and deployment checks.
  • Dashboard export issues (CSV/XLSX/PDF) and chart rendering questions.
  • History access credentials and transaction audit support.

Self-service checklist before contacting support

  • Confirm API health endpoint returns success.
  • Verify domain SSL status and deployment environment variables.
  • Retry in a private browser window after clearing cache.
